TCET Linux

TCET Linux is a GNU/Linux distribution developed in department of Electronics Engineering, TCET, which seeks to provide a Free and Open Source Solution to Operating System requirement of the institute. The distributiton is tailor made to suite the needs of the college's academic activities and work hand in hand with faculties and students.

Features

Easy to Install

Calamares installer makes it easier to install

Themed Desktop Environment

Users wanting to get started immediatly can choose the themed editions which have themed desktop enviroments.

Graphical environment

Ready to use graphical environments. Users can start using Arch Linux out of the box, with the themed editions which come with themes, settings and software.

For Intel/AMD computers

Hardware tested on team blue, and team red hardware.

Software to do practicals

Monthly rolling release schedule, just like mainline Arch Linux with new ISOs on first of every month.

Easy to multi-boot

Easily dual-boot/triple boot with Windows and/or other linux distributions.

Encryption using LUKS

All editions come with encryption using LUKS. USers can easily ecrypt drives during installtion.


Manual

TCET Linux developers provide detailed instructions for installation and use of TCET Linux.


Hardware Required

Here are the minimum system requirements, for TCET Linux.

  • Any 64-bit (x86_64) Intel/AMD APU
  • Min 1GB RAM
  • At least 5GB HDD/SSD space

To make bootable USB on Windows, use Balena Etcher. We recommend you to read the full manual to understand instructions on installation and usage.
